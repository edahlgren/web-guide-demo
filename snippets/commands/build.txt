  build

  Rebuild the source code


  Quick reference
  ===============

   Build sources using the default build configuration

     $ demo build
     
   Build sources cleanly, removing object files and old binaries first

     $ demo build --clean
     
   View stdout and stderr build logs

     $ cat build.log
     $ cat build.error
     

  Ways to build
  =============

   Use a preconfigured build

     $ demo build <configuration>
     
   Manually execute the commands produced by

     $ demo build <configuration> --dryrun
     

  Configured builds
  =================

   Build the demo with gcc

     $ demo build default
     
   Build like default at optimization level 3

     $ demo build optimized
     
   Build like default and include debug symbols

     $ demo build debug
     

  Files
  =====

   /root/src/acotsp/Makefile  Contains commands for building the acotsp binary
   /root/src/acotsp/acotsp    The acotsp binary, the output of a successful build
