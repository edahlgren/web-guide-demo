  run

  Run the demo


  Quick reference
  ===============

   Run the demo using the default configuration

     $ demo run
     
   Run the demo using command-line options (see below)

     $ demo run -- --opt1 arg1 --opt2 --opt3
     
   View stdout and stderr logs

     $ cat run.log
     $ cat run.error
     

  Summary
  =======

   Solve the traveling salesman problem using artificial ants

   Input
   
     TSPLIB file  Contains points and distances between points
   
   Output
   
     custom file  Contains statistics about the best solutions found
     custom file  Contains comparative statistics about all trial runs
     CSV file     Internal, unlabeled statistics
   

  Ways to run
  ===========

   Use a configured run

     $ demo run <configuration>
     
   Manually execute the commands produced by

     $ demo run <configuration> --dryrun
     
   Use your own command-line options

     $ demo run -- [options...]
     

  Configured runs
  ===============

   Find the shortest roundtrip through the cities of Djibouti

     $ demo run djibouti
     
   Find the shortest roundtrip through the cities of Qatar

     $ demo run qatar
     
   Find the shortest roundtrip through the cities of Luxembourg

     $ demo run luxembourg
     

  Command-line options
  ====================

   Choose one: Type of artificial ant colony to use
   
     --mmas, -x         true            MAX-MIN Ant System
     --as, -u           false           Basic Ant System
     --eas, -v          false           Elitist Ant System
     --ras, -w          false           Rank-based Ant System
     --acs, -z          false           Ant Colony System
     --bwas, -y         false           Best-worst Ant System
   
   Choose any
   
     --tsplibfile, -i   (none)          Path to a TSPLIB formatted file containing a 
                                          graph of points and distances between points
     --tries, -r        10              Number of independent trials
     --tours, -s        (none)          Number of steps in each trial
     --time, -t         10              Maximum time for each trial in seconds
     --seed             (current time)  Seed for the internal random number generator
     --optimum, -o      1               Stop if solution is better or equal to this 
                                          number (lower is better)
     --ants, -m         25              Number of ants (i.e. colony size)
     --nnants, -g       20              Number of ants in a neighborhood
     --alpha, -a        1               Influence of pheromone trails
     --beta, -b         2               Influence of heuristic information
     --rho, -e          0.5             Pheromone trail evaporation rate
     --q0, -q           (none)          Probability of finding the best solution
     --elitistants, -c  (none)          Number of elitist ants (with --eas only)
     --rasranks, -f     (none)          Number of ranks (with --ras only)
     --localsearch, -l  3               Configure local search
     --nnls, -k         20              Number of ants in a neighborhood for local 
                                          search
     --dlb, -d          true            Use don't look bits in local search
     --quiet            (none)          Reduce output to a minimum
   

  Examples
  ========

   Use the Elitist Ant System algorithm to solve Djibouti

     $ demo run -- --eas -i /root/data/djibouti/dj38.tsp
     
   Spend 20 seconds solving Qatar

     $ demo run -- --time 20 -i /root/data/qatar/qa194.tsp
     
   Use 100 ants to solve Djibouti

     $ demo run -- --ants 100 -i /root/data/djibouti/dj38.tsp
     
   Use the Basic Ant System and allow each ant to share information with its
   nearest 40 neighbors to solve Luxembourg

     $ demo run -- --as --nnants 40 -i /root/data/luxembourg/lu980.tsp
