Ant Colony Search

The ant colony search demo solves the traveling salesman problem using an artificial colony of ants. The traveling salesman problem is to find the shortest roundtrip through a region, visiting every city and town. For example:

Luxembourg
634 cities
Nicaragua
2373 cities
Greece
9882 cities

In the demo, artificial ants act like the traveling salesmen. As they explore various paths, the ant share information with their nearest neighbors, so as a collaborative swarm they all quickly find the best path.

First time guide

Enter the demo

The easiest way to experiment with the demo is inside the "demo shell", a terminal environment with everything needed by the demo already installed. First install the demo command line tool. Then execute this command to enter the demo and get access to the shell: 

$ demo shell {{download_path}}

Try it

Try each of the configured run scripts to test what the demo can do. Then try changing the behavior of the demo by using different command line options and input data.

Dig deeper

Once you've seen what the demo can do, you can learn how it works by reading its documentation, research papers, and source code. You can also change the source code to test out new ideas.
