run

Run the demo

Quick reference

Run the demo using the default configuration

$ demo run

Run the demo using command-line options (see below)

$ demo run -- --opt1 arg1 --opt2 --opt3

View stdout and stderr logs

$ cat run.log
$ cat run.error

Summary

Solve the traveling salesman problem using artificial ants
___Input
TSPLIB fileContains points and distances between points
___Output
custom fileContains statistics about the best solutions found
custom fileContains comparative statistics about all trial runs
CSV fileInternal, unlabeled statistics

Ways to run

Use a configured run

$ demo run <configuration>

Manually execute the commands produced by

$ demo run <configuration> --dryrun

Use your own command-line options

$ demo run -- [options...]

Configured runs

Find the shortest roundtrip through the cities of Djibouti

$ demo run djibouti

Find the shortest roundtrip through the cities of Qatar

$ demo run qatar

Find the shortest roundtrip through the cities of Luxembourg

$ demo run luxembourg

Command-line options
___Choose one: Type of artificial ant colony to use
--mmas, -xtrueMAX-MIN Ant System
--as, -ufalseBasic Ant System
--eas, -vfalseElitist Ant System
--ras, -wfalseRank-based Ant System
--acs, -zfalseAnt Colony System
--bwas, -yfalseBest-worst Ant System
___Choose any
--tsplibfile, -i(none)Path to a TSPLIB formatted file containing a graph of points and distances between points
--tries, -r10Number of independent trials
--tours, -s(none)Number of steps in each trial
--time, -t10Maximum time for each trial in seconds
--seed(current time)Seed for the internal random number generator
--optimum, -o1Stop if solution is better or equal to this number (lower is better)
--ants, -m25Number of ants (i.e. colony size)
--nnants, -g20Number of ants in a neighborhood
--alpha, -a1Influence of pheromone trails
--beta, -b2Influence of heuristic information
--rho, -e0.5Pheromone trail evaporation rate
--q0, -q(none)Probability of finding the best solution
--elitistants, -c(none)Number of elitist ants (with --eas only)
--rasranks, -f(none)Number of ranks (with --ras only)
--localsearch, -l3Configure local search
--nnls, -k20Number of ants in a neighborhood for local search
--dlb, -dtrueUse don't look bits in local search
--quiet(none)Reduce output to a minimum

Examples

Use the Elitist Ant System algorithm to solve Djibouti

$ demo run -- --eas -i /root/data/djibouti/dj38.tsp

Spend 20 seconds solving Qatar

$ demo run -- --time 20 -i /root/data/qatar/qa194.tsp

Use 100 ants to solve Djibouti

$ demo run -- --ants 100 -i /root/data/djibouti/dj38.tsp

Use the Basic Ant System and allow each ant to share information with its nearest 40 neighbors to solve Luxembourg

$ demo run -- --as --nnants 40 -i /root/data/luxembourg/lu980.tsp
